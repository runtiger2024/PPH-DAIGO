新增商品

批量新增/更新商品

您可以透過上傳 Excel (.xlsx) 檔案來一次新增多筆商品，或根據商品 ID 更新現有商品。

步驟：
  1. 點此下載 Excel 範本檔案。
  2. 依照範本格式填寫您的商品資料（注意：標題有 * 為必填欄位）。
  3. 若要更新現有商品，請務必填寫第一欄的 `ID`；若要新增商品，請將 `ID` 欄留空。
  4. 儲存 Excel 檔後，透過下方的表單上傳。
